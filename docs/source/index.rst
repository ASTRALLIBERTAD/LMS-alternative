LMS Application Documentation
==============================

Welcome to the LMS (Learning Management System) Application documentation.

This system provides a comprehensive solution for managing assignments, students,
and submissions with Google Drive integration.

Features
--------

* Google Drive integration for file storage
* Assignment management and tracking
* Student submission handling
* File preview for multiple formats
* OAuth authentication
* Responsive UI with Flet
